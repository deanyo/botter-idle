Botter Authoring Portal

Browser-based editors for biomes, items, affixes, and tiles. Author content locally, export JSON, share back via GitHub issues / Discord.

Editors

live Tile Atlas Viewer

Browse all 6,945 DCSS tiles by category, biome, class, or subdir. Filter, search, click to copy paths. Read-only — pairs with the editors below.

 live Biome Editor

Visual editor for floor, wall, edge-overlay, and sigil sets per biome. Replace tiles by picker. Exports a complete biomes.json.

 live Item Editor

Slot-tabbed editor (1H sword / helm / armor / shield / boots / ring / amulet). Edit stats, drop weights, flavor tags, lore. Exports per-slot JSON slices.

 live Affix / Enchant Editor

Author new affixes (rarity-tiered stat bonuses) with a color picker, stat dropdown, and live preview card. Exports a complete affixes.json.

Sharing what you build

Each editor has an ⬇ Export button that downloads JSON. To share your design:

  1. Hit Export and save the file locally.
  2. Open a GitHub issue with the JSON contents (or a Discord post if linked from the project).
  3. If it gets accepted, your file lands in project/data/ next release.

No backend, no auth, no upload — your work stays local until you choose to share it.

Working locally

Clone the repo, then serve tools/ over HTTP so the editors can fetch() data files (browsers block file:// fetch):

git clone https://github.com/deanyo/botter-idle.git
cd botter-idle
python3 -m http.server 8080
# then open http://localhost:8080/tools/
